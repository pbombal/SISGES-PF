# SISGES-PF - Sistema de Gestión de Planes Formativos

Este sistema permite:
- Verificar nuevos planes formativos del catálogo SIPFOR
- Descargar automáticamente los planes en PDF
- Consolidar la información de los módulos desde los PDFs
- Generar una reportería visual con Streamlit

## Ejecución
1. Instala dependencias:
   pip install -r requirements.txt

2. Ejecuta el sistema:
   python lanzar.py

## Estructura esperada de carpetas
- /data/planes_descargados
- /data/consolidado
